```bash
curl "http://localhost:8000/recommend/also-bought/P004?n=5"
```

**Response:**
```json
{
  "product_id": "P004",
  "count": 5,
  "also_bought": [
    {"rank": 1, "product_id": "P005", "product_name": "Ergonomic Mouse Wireless",  "confidence": 0.8310, "lift": 2.87, "score": 2.384970},
    {"rank": 2, "product_id": "P022", "product_name": "Mousepad XL Gaming",        "confidence": 0.7540, "lift": 2.61, "score": 1.967940},
    {"rank": 3, "product_id": "P007", "product_name": "Monitor Light Bar",          "confidence": 0.6820, "lift": 2.34, "score": 1.595880},
    {"rank": 4, "product_id": "P017", "product_name": "USB Hub 7-Port",             "confidence": 0.6130, "lift": 2.11, "score": 1.293430},
    {"rank": 5, "product_id": "P008", "product_name": "Cable Management Kit",       "confidence": 0.5410, "lift": 1.87, "score": 1.011670}
```

**Rule interpretation:**
> "83.1% of customers who bought a **Mechanical Keyboard** also bought an
> **Ergonomic Mouse**. This combination occurs 2.87× more often than chance."

## 4 — Basket Completion

**Request:**
```bash

     -H "Content-Type: application/json" \
     -d '{"product_ids": ["P004", "P005"], "n": 3}'
```

**Response:**
```json
{
  "product_id": "basket",
```
